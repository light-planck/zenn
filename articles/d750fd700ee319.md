## はじめに

JavaScriptの基本的なデータ型であるプリミティブ型と参照型について解説します。
それらの違いや振る舞いを理解することで、適切にデータを扱うことができるようになります。

## 対象読者

- JavaScriptを勉強し始めた方
- プリミティブと参照についてあいまいな方

## JavaScriptのデータ型の種類

JavaScriptのデータ型は大きく分けてプリミティブ型と参照型の2種類あります。

- プリミティブ型
  - 数値型 (number)
  - 長整数型 (bigint)
  - 文字列型 (string)
  - 論理型 (boolean)
  - シンボル型 (symbol)
  - 特殊型 (null, undefined)
- 参照型
  - 配列 (array)
  - オブジェクト (object)
  - 関数 (function)

配列、オブジェクト、関数は参照型でそれ以外はプリミティブ型と覚えるといいです。

## 違い

変数に格納されるものが違います。
プリミティブ型の場合はその値（リテラル）を、参照型は参照値（メモリのアドレス）を格納します。

## 具体例

- プリミティブ型

```js
// プリミティブ型
let n = 100;
console.log(n);  // output: 100
let m = n;
n = 0;
console.log(m);  // output: 1

let s = "abc";
console.log(s);  // output: "abc"
let t = s;
s = "none";
console.log(t);  // output: "abc"
```

- 参照型

```js
// 参照型
let obj1 = {
  title: "javascript",
  page: 400,
};
console.log(obj1);  // output: { title: 'javascript', page: 400 }

let obj2 = obj1;
obj1.title = "java";
console.log(obj2);  // output: { title: 'java', page: 400 }
```
